🎵 France Top 50 Spotify Playlist Analytics Dashboard

📌 Project Overview

This project analyzes the **France Top 50 Spotify Playlist** to understand audience preferences, content compliance, artist popularity, and album format trends. An interactive dashboard was developed using Streamlit to help users explore the dataset through filters, KPIs, and visualizations.



🎯 Objectives

- Analyze artist popularity and playlist trends.
- Compare album and single releases.
- Study explicit vs. non-explicit songs.
- Measure song duration and popularity.
- Build an interactive dashboard for business insights.

---

🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Plotly
- Matplotlib
- Seaborn
- Streamlit
- Git & GitHub

---
📂 Dataset

The dataset contains information about the France Top 50 Spotify Playlist, including:

- Song Name
- Artist
- Popularity
- Album Type
- Duration
- Explicit Content
- Total Tracks
- Album Cover URL

---

📊 Dashboard Features

- Interactive Sidebar Filters
- KPI Cards
- Top Artists Analysis
- Top Songs Analysis
- Popularity Distribution
- Explicit vs Non-Explicit Songs
- Album Type Distribution
- Song Duration Analysis
- Correlation Heatmap
- Download Filtered Dataset

📈 Key Insights

- Identified the most popular artists in the playlist.
- Compared album and single releases.
- Analyzed explicit content distribution.
